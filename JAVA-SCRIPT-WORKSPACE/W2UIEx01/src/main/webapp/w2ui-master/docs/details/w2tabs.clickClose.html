Called when user clicks close button of the tab.
clickClose(id, [event])
id string, id of the tab
event object, DOM Event object
Returns undefined.

Description

This method is called when user clicks on the close button of the tab. It is used internally, but you can use it to emulate clicks. It will animate closing of the tab. This method emits .onClose event.
