Indicates if column vertical scsoll (virtualization) is enabled.
Boolean, default = false
This property only applies if your grid has many colunns. By default column virtualization is abled, menaing only columns in the view will be rendeder in DOM. This is somewhat experimental feature. If you do not have large number of columns, it might be better to set .disableCVS to true.
This property can be set during object creation: Or any time during run-time:
