Called when user types within a form field.
onInput = function(event)
This event is triggered each time user types within a form field. In the event object you can find the new value of the field. This event is usually triggered before the onChange event fires. You can add event listener during the object creation: or any time during run-time: The event handler is called before the default action of the event is triggered. You can cancel the default action by calling event.preventDefault(). To perform an action after the event is fully processed, define event.onComplete function.
See events page in utilities for more details.
