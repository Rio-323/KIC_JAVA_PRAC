Called when user selects a columng.
onColumnSelect = function(event)
If user just clicks on the column, it will trigger sorting. However, if user holds cmd/ctrl/shift key and clicks on the column, he triggers column selection feature. It will receive one argument event that will contain information about the column.
You can add event listener during the object creation: or after the object has been created: Your event listener can be executed before or after the event has been processed. You can also cancel the default behavior. See Events page in utilities.
