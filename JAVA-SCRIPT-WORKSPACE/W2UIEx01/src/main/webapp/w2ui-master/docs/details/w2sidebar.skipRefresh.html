Indicated if to refresh during add/remove of nodes
Boolean, default = false
If you need to add a lot of nodes, it might be useful not to refresh during each operation. Set .skipRefresh = false to skip refresh when new nodes are added/deleted with the following functions
  • w2sidebar.add()
  • w2sidebar.insert()
  • w2sidebar.remove()
  • w2sidebar.set()
