Defines HTML in the right corner of the toolbar.
Array of strings, default = ['']
You can define a custom HTML to appear in the right corner of the toolbar if you set .right property. If it is a single string, it will be converted to an single cell array. You can render buttons in the right corner of the toolbar if you add a toolbar item of type = spacer. If you add a spacer, all of the toolbar items that follow will be aligned to the right, but right text will still be on the right from them.
